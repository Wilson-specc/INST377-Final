What Brought This Idea?

New Pokémon TCG players often don’t know how many different card variations exist for their favorites. That makes collecting chaotic — until now.

What This Search Does

Using the Pokémon TCG API, this app fetches and shows all card variations for any Pokémon name you type. You get visuals, data, and plenty of collection clarity.

Tools

  • HTML, CSS, JavaScript
  • Pokémon TCG API (external data source)
  • Fetch API for retreiving card data
  • Supabase for backend memory
  • Chart.js for visual stats
